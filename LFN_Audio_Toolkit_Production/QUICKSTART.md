# Quick Start Guide

Get up and running with LFN Audio Toolkit in 5 minutes!

## Installation

```bash
# 1. Clone repository
git clone https://github.com/yourusername/lfn-audio-toolkit.git
cd lfn-audio-toolkit

# 2. Run setup
python setup.py
```

## Usage Examples

### Analyze Audio Files
```bash
python src/lfn_batch_file_analyzer.py "C:\path\to\audio"
```

### Real-Time Monitoring
```bash
python src/lfn_realtime_monitor.py
```

### Long Recording
```bash
python src/long_duration_recorder.py
```

## Output Files

Results are saved to `outputs/`:
- **CSV files**: Spreadsheet-compatible data
- **Spectrograms**: Visual frequency analysis
- **Trends**: Time-series plots
